# Черновик цели на четыре недели

Статическая двухуровневая страница с короткой практикой для формирования первого черновика цели.

Ответы сохраняются только в `localStorage` браузера. Страница не отправляет введённые данные на сервер и не требует регистрации.

## Структура

- `index.html` — страница практики;
- `assets/css/styles.css` — стили;
- `assets/js/app.js` — интерактивность и локальное сохранение;
- `assets/img/` — изображения;
- `assets/pdf/` — памятка в PDF.

Для локального просмотра достаточно открыть `index.html` в браузере.
